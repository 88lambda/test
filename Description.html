To start using ONLYOFFICE Docs with Nextcloud or ownCloud, you have to install the respective app and enable the ONLYOFFICE plugin in either app.

The address to the document server has to be configured inside the plugin. Pleaes note that the parameter documentserver is the name (FQDN, without a port number) of the server with the ONLYOFFICE Docs installed. The address must be accessible for the user browser and from the respective Nextcloud or ownCloud server. The server address must also be accessible from ONLYOFFICE Docs for operating correctly. 

 https:///onlyoffice-documentserver

Configuration of ONLYOFFICE plugin in Nextcloud

  • Login to Nextcloud as user Administrator.
  • Goto Apps → Office & Text → enable ONLYOFFICE.
  • Goto Admin settings and look for ONLYOFFICE. There enter the address above to connect the ONLYOFFICE document server and click on Save.

Configuration of ONLYOFFICE plugin in ownCloud

The ONLYOFFICE plugin in ownCloud is already pre-installed in ownCloud, but deactivated by default. If you install ONLYOFFICE on the same system as ownCloud, the plugin will be activated in ownCloud automatically.

You can check and activate the app manually following theses steps:

  • If ownCloud is not installed yet, goto the App Center, select ownCloud and install it.
  • Login to ownCloud as admin user (username owncloud).
  • Goto Settings → Admin Apps → Show disabled Apps → [Enable] ONLYOFFICE
  • Goto Settings → Admin → Additional → ONLYOFFICE: Check the setting for Document Editing Service. By default it is set to the UCS host, where ownCloud is installed.
The ONLYOFFICE plugin in ownCloud is already pre-installed. It will be activated as soon as ONLYOFFICE is installed on the same system, where ownCloud is installed.

Details

ONLYOFFICE Docs is a web-based document, spreadsheet, presentation, form, and PDF processing, highly compatible with Microsoft Office and OpenDocument file formats. This app offers a powerful feature set that enables you to view, edit and co-author all kinds of Office documents:

  • Tabbed interface: Tools are organized in convenient tabs for better navigation.
  • Wide range of formatting features: Add links, tables and charts, insert and format pictures, work with auto-shapes, equations or text objects, create a bulleted or numbered list, etc.
  • Broad collaboration capabilities: Comment your documents, interact with your co-authors via a built-in chat, switch between two co-editing modes (fast or strict), enable the review mode, track changes, check the version history.
  • Add-on collection: Translate text, work faster using AI helpers, edit your images with Photo Editor, embed YouTube videos and more.

ONLYOFFICE Docs offers the support of all the popular formats: DOC, DOCX, TXT, ODT, RTF, ODP, EPUB, ODS, XLS, XLSX, CSV, PPTX, HTML, DOTX, XLTX, POTX, OTT, OTS, OTP, PDF and PDF-A.

About this app

This app is a community version of ONLYOFFICE Docs and is limited to 20 simultaneous connections (number of documents that can be edited ad the same time) and is recommended for teams of up to 20 users.

Your documents are stored in file sync and share solutions like Nextcloud, ownCloud or Seafile. ONLYOFFICE does not store user document data.

If you want to increase the number of sessions, please obtain ONLYOFFICE Docs Enterprise Edition via 'Buy now' button and install the app afterwards via the App Center.
